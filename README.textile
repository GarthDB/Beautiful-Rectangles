h2. Beautiful Rectangles: UX Done Right

Beautiful Rectangles is simple set of templates and libraries to make it a little easier to create user experience design deliverables.

Robotlegs enables:

* Personas
* Sitemaps
* Wireframes
* Click-Through Prototypes

"Official Beautiful Rectangles Website":http://www.beautifulrectangles.com

h3. Requirements

The library is designed to work with what a user experience designer should already have:

"Adobe CS5":http://www.adobe.com/products/creativesuite/
* Illustrator CS5
* InDesign CS5
* Photoshop CS5
* Fireworks CS5

* Omnigraffle (coming soon)
